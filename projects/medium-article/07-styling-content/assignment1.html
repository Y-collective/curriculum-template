Minden tartalom szerepel az oldalon, amit szerettünk volna, logikus egységekbe szervezve. A tartalmi egységek megformázása áll még előttünk.

A HTML tehát leírja az oldal tartalmát, a CSS határozza meg annak stílusát.

Kattints rá a JSBin CSS paneljére (ha még nincs nyitva), ide fogjuk írni a kódot.

A CSS kód a következőképpen néz ki:

  
    kivalaszto {
      tulajdonsag: ertek;
    }

A kiválasztó (selector) megmondja, hogy mire legyen érvényes a szabály. A szabályon belül pedig különböző tulajdonságoknak (property) adhatunk értéket (value).

+1 tipp: Mindig figyelj a programozásnál a zárójelek és a különböző pontok, vesszők használatára. Ezek jelzik a programnyelv értelmezésénél a dolgok kezdetét és végét.

Első lépésként állítsd be, hogy a legnagyobb címsor nagyobb betűméretű legyen, és fekete helyett szürke színű. Másold át a következő kódrészletet:

  
    h1 {
      font-size: 36px;
      color: rgba(0,0,0,.8);
    }

A h1 mondja meg, hogy az összes ilyen tag-be csomagolt HTML tartalomra legyen érvényes mindaz a szabály, ami a kapcsos zárójelek között következik.

A font-size állítja a betűméretet, itt a megjelölt 36px nagyságúra.

A color szabja meg a betűk színét. A szín többféleképpen megadható, a példában az rgba formátumot választottuk, ahol:

  
    rgba(piros, zöld, kék, átlászóság)

ahol a piros, zöld, és kék értékek 0 és 255 közötti számok, az átlátszóság pedig egy 0 és 1 közötti tört szám, tizedesponttal. Pl. 0.78. (A tört számoknál a CSS csak pontot fogad el, tehát pl. a 0,12 érték helytelen lenne.) Ha a tört szám egészrésze nulla, javasolt azt elhagyni (ahogy a példában is látod).

+1 tipp: Ezen a színkeverő oldalon közvetlenül az R (red), G (green), B (blue) és A (alpha, átlátszóság) összetevőket állíthatod, így biztosan megérted, hogyan keverednek a színek. Ha még nem vagy otthon az RGB-színkeverésben, játssz kicsit a csúszkákkal, hogy jobban megmaradjon. A jobb oldalon pedig ki tudod másolni a CSS kódot.

Pro tipp: A példában valójában fekete színt használunk, 80%-os átlátszósággal, emiatt látszik a fehér háttéren szürkének. Ez egy jó gyakorlat, abban segít a későbbiekben, hogy a szövegünk az oldal háttérszínének megváltoztatása esetén is jól mutasson.
