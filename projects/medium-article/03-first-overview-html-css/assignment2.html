Most pedig kiszínezzük az oldalt, CSS segítségével.

Jegyezd meg, hogy mindig a HTML csinálja az oldal tartalmát, a CSS pedig a színezést, méretezést, betűtípusokat.

A JSBin közepén van egy “CSS” gomb, ami kinyit/becsuk egy középső panelt. Nyomd meg, hogy kinyíljon a CSS szerkesztő. Ezután a CSS szerkesztőbe gépeld be a következőt:


  html { background-color:red; }

+1 tipp: A kapcsos zárójelet Windowson, magyar billentyűzeten a jobb oldali Alt + B ill. Alt + N billentyűkombinációkkal, a pontosvesszőt Alt + , -vel tudod beírni.

Próbálkozz más színekkel is! Az informatikai feladatoknál fontos, hogy ne egy unalmas iskolai példának fogd fel őket, hanem játékoknak. Mint a legó. Játssz vele, próbálgasd, rontsd el. Ctrl + Z -vel szinte mindent vissza lehet csinálni :)

+1 tipp: rengeteget tanulsz azzal, ha játszol, próbálkozol, élvezed a feladatot és nem félsz elrontani, mint ha csak utasításokat követsz. Ha sikerül ezt fejben tartanod, gyorsan fog menni a tanulás.
