A cikkben a kép alatt szerepel egy képaláírás is. Ezt a <figcaption> tag segítségével tudod hozzáadni az oldal tartalmához:

  
    <figcaption>Learning to code at a Free Code Camp Toronto coffee-and-code.</figcaption>

A tag-ek egymásba ágyazhatók. Annak érdekében, hogy könnyebben tudd együtt kezelni a képet és a képaláírást, csomagold be azokat egy <figure> tag segítségével (ez az ábrák "becsomagolására" használt HTML tag), így:

  
    <figure>
      <img src="http://i.imgur.com/RC2UGg8.jpg" alt="coders">
      <figcaption>Learning to code at a Free Code Camp Toronto coffee-and-code.</figcaption>
    </figure>

Pro tipp: Mindig ügyelj arra, hogy tag-ek egymásba ágyazásakor a később megnyitott tag-eket zárd be előbb, sorban. Tehát <figure><figcaption></figcaption></figure> és nem <figure><figcaption></figure></figcaption>
